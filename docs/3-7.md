# 검색할 때 필요없는 HTML 태그 제거하기(html_strip)
## ✅ 검색 시 필요없는 HTML 태그 제거하기
```
// 기존 인덱스 삭제
DELETE /boards

// 인덱스 생성 + Custom Analyzer 적용
PUT /boards
{
    "settings": {
        "analysis": {
            "analyzer": {
                "boards_content_analyzer": {
                    "char_filter": ["html_strip"],
                    "tokenizer": "standard",
                    "filter": ["lowercase"]
                }
            }
        }
    },
    "mappings": {
        "properties": {
            "content": {
                "type": "text",
                "analyzer": "boards_content_analyzer"
            }
        }
    }
}

POST /boards/_doc
{
    "content": "<h1>Running cats, jumping quickly - over the lazy dogs!</h1>"
}

GET /boards/_search
{
    "query": {
        "match": {
          "content": "running"
        }
    }
}

GET /boards/_analyze
{
    "field": "content",
    "text": "<h1>Running cats, jumping quickly - over the lazy dogs!</h1>"
}
```

## ✅ 정리
HTML 태그가 포함된 데이터를 검색에 사용될 때는 character filter로 `html_strip` 적용을 고려해보자.
