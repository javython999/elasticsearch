# Elasticsearch 설치

## ✅ Docker를 활용한 설치
1. compose.yml 작성
```yml
services:
  elastic:
    image: docker.elastic.co/elasticsearch/elasticsearch:8.17.4
    ports:
      - 9200:9200
    environment:
      - discovery.type=single-node
      - xpack.security.enabled=false
      - xpack.security.http.ssl.enabled=false
```
2. compose 실행
```shell
docker compose up
```
