# 검색한 키워드를 하이라이팅 처리하고 싶을 때(highlighting)
## ✅ 검색한 키워드를 하이라이팅 처리하고 싶을 때(highlighting)
```
GET /boards/_search
{
    "query": {
        "multi_match": {
            "query": "엘라스틱서치 적용 후기",
            "fields": ["title^2", "content"]
        }
    },
    "highlight": {
        "fields": {
            "title": {
                "pre_tags": ["<mark>"],
                "post_tags": ["</mark>"]
            },
            "content": {
                "pre_tags": ["<b>"],
                "post_tags": ["</b>"]                
            }
        }
    }
}
```
