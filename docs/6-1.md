# 검색 키워드가 포함된 데이터를 조회하고 싶을 때(match)
## ✅ 검색 키워드가 포함된 데이터를 조회하고 싶을 때(match)
* match 쿼리는 검색 키워드가 포함된 모든 도큐먼트를 조회한다.
* match 쿼리는 text 타입의 필드에서만 사용하는 쿼리다.

```
DELETE /boards

PUT /boards
{
    "mappings": {
        "properties": {
            "title": {
                "type": "text"
            }
        }
    }
}

POST /boards/_doc
{
    "title": "편의점 과자 내돈내산 후기"
}

GET /boards/_search
{
    "query": {
        "match": {
          "title": "편의점 후기"
        }
    }
}
```
