# 페이지네이션(Pagination), 정렬(sort)
## ✅ 페이지네이션(Pagination)
```
DELETE /boards

PUT /boards
{
    "mappings": {
        "properties": {
            "title": {
                "type": "text",
                "analyzer": "nori"
            },
            "likes": {
                "type": "long"
            }
        }
    }
}

POST /boards/_doc
{
    "title": "1번 글",
    "likes": 12
}

POST /boards/_doc
{
    "title": "2번 글",
    "likes": 35
}

POST /boards/_doc
{
    "title": "3번 글",
    "likes": 1
}

POST /boards/_doc
{
    "title": "4번 글",
    "likes": 11
}

POST /boards/_doc
{
    "title": "5번 글",
    "likes": 72
}

POST /boards/_doc
{
    "title": "6번 글",
    "likes": 10
}

POST /boards/_doc
{
    "title": "7번 글",
    "likes": 5
}


GET /boards/_search
{
    "query": {
        "match": {
            "title": "글"
        }
    },
    "size": 3,
    "from": 0
}

GET /boards/_search
{
    "query": {
        "match": {
            "title": "글"
        }
    },
    "sort": [
        {
            "likes": {
                "order": "desc"
            }
        }
    ]
}
```
* `size`: 한 페이지에 불러올 데이터 개수(SQL의 LIMIT와 동일)
* `from`: 몇 번째 데이터부터 불러올 지 (SQL의 OFFSET과 동일, 0부터 시작)
